JoelCraft.Dev - Sitio de ejemplo
Archivos:
- index.html
- styles.css
- script.js
- assets/logo.png

Cómo usar:
1. Abrir el proyecto en Visual Studio Code.
2. Abrir index.html en el navegador (doble clic o con Live Server).
3. Edita textos, cuentas bancarias y enlaces en index.html.

Animación:
- El logo animado usa un canvas con partículas y la imagen que proporcionaste en assets/logo.png.
